# Screenshots — EduSphere

All project screenshots are stored here. They are captured during development sessions as visual verification evidence.

## Naming Convention

`scan-<feature>[-<variant>].png` — Playwright or manual browser scans
`step<N>-<description>.png` — Step-by-step flow verification screenshots
